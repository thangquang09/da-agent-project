# AGENTS.md - docs/

Mục tiêu của thư mục `docs/` là giảm context load cho coding agent.

## Routing nhanh
- Nếu cần tài liệu ưu tiên cho chủ repo: mở `docs/thangquang09/AGENTS.md`.
- Nếu cần tài liệu nghiên cứu/lý thuyết/benchmark: mở `docs/research/AGENTS.md`.
- Không quét toàn bộ `docs/` nếu không cần thiết.

## Quy ước
- Mỗi folder con phải có `AGENTS.md` mô tả phạm vi và file cần đọc.
- Đặt tên file rõ ràng theo chủ đề + ngày nếu là ghi chú nghiên cứu.
- Ưu tiên cập nhật file ở đúng folder thay vì tạo file md ở root repo.
